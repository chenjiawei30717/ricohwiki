 ![image.png](http://172.31.68.75:8090/upload/2022/02/image-6c922d6d2eca48aab7245650cf4b636d.png)
## 1.3、数字计划与排程
摆脱传统纸面与电子表格的管理方式，除一般生产加工计划外，将调试、保养等项目融合到计划中，通过可视化即时、清晰的根据管理整个计划进度，画面操作简单、高效。
![image.png](http://172.31.68.75:8090/upload/2022/02/image-0e4d53537b6247809a58df99588c92d7.png)
## 1.4、工位数字工作站
员工岗位别“数字工作站”，集成关联作业信息与操作入口，让员工更加容易的了解岗位职能,作业内容，避免了传统操作入口的不断切换。
## 1.5、品质管控与追溯
集检查规格数据管理、检查结果记录、SPC管控、质量报表等的检查系统，同时结合生产执行及物料使用实现品质追溯功能。

## 3.1、工厂建模
- 生产过程中主要对象参数的数字化信息。
|对象|内容|
|-------|-------|
|部品|部番、后缀、类型、箱型、箱数、内包装型号、内包装数、备注(机型等)|
|模具|模具号、类型、测量治具、供应商、流道形式、取数、型共通标识|
|设备|设备类型、固定资产编号、设备厂家、设备型号、设备编号、保养/段取时间|
|工艺|部品、设备、模具、成型周期、回收料比例、部品重量、浇口重量、自动标识|
|材料|材料号、材质、等级、色号、代码|
|仓位|区域号、仓位号|
|荷姿|部番、包装箱型、垫板号、垫板数、隔档数、内包装型号、内包装数、捆包方式|
|类目|工程不良类型|
|点检|点检类型、分类、顺序、点检项|
- 建模数据新建、查询、删除、修改，可以通过Excel文件一括上传，同时支持已有数据的下载。

## 3.3、物料管理
全流程精益化管理，通过条码化，材料纳入、入库、出库、出库、领料、加料利用移动PDA作业，
便捷、高效，材料原流程在库状态可视化。         
### 库存作业：
原材料点收→入库→出库→领料(交接)→加料→干燥的全
#### 原材料点收
采购订单点收，利用原材料条码标签和PDA设备快速完成点收作业。
![image.png](http://172.31.68.75:8090/upload/2022/02/image-d95ec3d3421a4664b3d090163c9a58a6.png)
